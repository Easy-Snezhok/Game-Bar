# Game-Bar

Интерфейс игрового магазина с кастомной логикой и адаптивной версткой, разработанный с нуля — от дизайна до финального кода.

## 🛠 Технологический стек
* **Design:** Figma (UI/UX, прототипирование, UI-kit, адаптивные сетки)
* **Frontend:** HTML5 (семантическая разметка), CSS3 (чистый код с нуля), JavaScript (ES6+)
* **Инструменты:** VS Code, Git, GitHub, Chrome DevTools

## 🚀 Ключевые фичи проекта
1. **Кастомный JS-слайдер:** Написан без сторонних библиотек на чистом `requestAnimationFrame` с использованием кубического сглаживания.
2. **Клиентская мультиязычность:** Реализована система локализации страниц на стороне клиента через `data-атрибуты` и JavaScript.
3. **Оптимизация производительности:** Настроена предзагрузка (`preload`) тяжелых графических элементов для предотвращения задержек при отображении первого экрана.
